# Feature: Project Details 

## Goal 

implement `project details.html` 

## Designs 

![target layout](./assets/images/spec.png) 

## Work

- Describe the UI/UX clearly and specifically 
  It should present my different projects as cards with details under the main image. When clicked, the project will appear on the side (like the character selection lab).
- What elements exist? What container elements exist?
  Images, text and buttons. There will also be a page to see another on in it.
- How do users interact? What is the hover behavior? 
  The user should click on the different cards to see more details about the projects. If they hover over the cards, it should slightly move.

## Deliverables

- What does “done” look like? 
  A design like a card selection in a futuristic style where the user can have more details about the different projects.
- Be concrete and testable
